# Érase una vez Quiz
![image](https://github.com/IsaacGonzalez1/EraseQuiz/assets/72462257/d2916491-2d03-4255-926f-a37d4c008ac6)

Bienvenido a "Érase una vez Quiz", una plataforma educativa interactiva diseñada específicamente para niños pequeños, con el objetivo de proporcionar una experiencia de aprendizaje atractiva y enriquecedora. Basándonos en la icónica serie "Érase una vez", nuestra plataforma ofrece una forma única de explorar temas educativos importantes.

## Descripción Detallada

"Érase una vez Quiz" integra videos educativos interactivos en tres áreas clave: el cuerpo humano, la historia de la humanidad y el espacio. Estos videos están diseñados para ser tanto informativos como entretenidos, fomentando la curiosidad y el aprendizaje en los niños.

### Características Principales:

- **Videos Educativos Interactivos**: Videos sobre el cuerpo humano, la historia de la humanidad y el espacio, diseñados para ser atractivos y educativos.
- **Temas Variados**: Cada sección se enfoca en un tema distinto, ofreciendo una amplia gama de conocimientos.
- **Diseño Infantil y Atractivo**: Una interfaz colorida y amigable que atrae y mantiene el interés de los niños.
- **Accesibilidad y Facilidad de Uso**: Diseñado para ser accesible y fácil de usar para los niños.

## Instalación y Configuración

Instrucciones para instalar Erase Quiz: Para ello habrá una única manera:
- Instalando nuestro repositorio en su dispositivo y visualizandolo con cualquier IDE (Visual Studio Code) -> Plugin: Live Server
